Visualization for sorting algorithms

Enter the size of the array in the textbox and press enter to change the size of the array
Click on each method for visualization of each method.

Included methods:
1. Selection sort
2. Insertion sort
3. Merge sort
4. Quick sort
5. Bubble sort
6. Shell sort
7. Comb sort
8. Counting sort
9. Bucket sort
10. Radix sort

Notice that the time spent in this visualization cannot be used to compare the runtime of each algorithm.
